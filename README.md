# Trick-Play
This thing is used to play videos in the `videos_ids.py` on a loop cleanly through the firefox api.

## Installation
### Pre-Requisites
  - Ubuntu
  - Firefox
  - Python3

### Configurations
``` bash
  ./hello.bash BLM
  firefox -P BLM
```

Then install an add-on called `kill-tab`

### Run Things
``` bash
  python3 ./trick-play.py
```
